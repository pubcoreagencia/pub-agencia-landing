# PUB Agência Landing Page

Landing page oficial da agência PUB, construída como site estático pronto para publicação.

Direção atual: agência de publicidade, tráfego pago, SEO, conteúdo de busca e landing pages.

## Estrutura

- `index.html`: marcação da landing page
- `styles.css`: identidade visual, responsividade e animações
- `app.js`: conteúdo editável, interações, menu mobile e cena orbital em canvas

## Edição rápida

No topo de `app.js`, ajuste:

- `CONTACT_URL`
- `services`
- `workedBrands`
- `territories`
- `testimonialHighlights`
- `processSteps`

Os depoimentos não foram preenchidos automaticamente porque o destaque do Instagram não pôde ser acessado por automação. Insira apenas depoimentos reais em `testimonialHighlights`.
